# GloStack Website

Landing page for the GloStack iOS app, hosted on GitHub Pages at **https://glostack.app**.
Built from the high-fidelity "App Download" design handoff: two angled phone mockups,
four floating illustration cards, and an App Store badge on a warm cream panel.

## Files
- `index.html` — the App Download landing page
- `privacy.html` — privacy policy (replace placeholder text before launch)
- `styles.css` — all styling (design tokens recreated from the handoff)
- `assets/` — images / favicon / social image
- `CNAME` — tells GitHub Pages to serve the site at glostack.app (do not delete)

## Placeholders to replace with real assets
These match the `data-placeholder` labels in `index.html`:

| Placeholder | What it is |
|---|---|
| Screen (left) | App screenshot inside the left phone |
| Screen (right) | App screenshot inside the right phone |
| Orange / Yellow / Blue / Purple | Floating feature/illustration cards |

Also replace:
- The **App Store link** (`href="#"` on the `.store-badge` in `index.html`)
- The brand wordmark / dot if you have a real logo
- Real privacy policy text in `privacy.html`
- Favicon at `assets/favicon.png` and social image at `assets/og-image.png`

Plain HTML/CSS — edit, commit, push. Changes go live within a minute or two.
