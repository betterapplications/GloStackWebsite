# GloStack Website

Landing page for the GloStack iOS app, hosted on GitHub Pages at **https://glostack.app**.

## Files
- `index.html` — main landing page
- `privacy.html` — privacy policy (replace placeholder text before launch)
- `styles.css` — all styling
- `assets/` — images, App Store badge, favicon
- `CNAME` — tells GitHub Pages to serve the site at glostack.app (do not delete)

## Editing
It's plain HTML/CSS — open the files, change the text, save, commit, push. Changes
go live within a minute or two.

## Things to replace before launch
- App Store download link (`href="#"` in `index.html`)
- Real screenshot at `assets/screenshot.png`
- Favicon at `assets/favicon.png` and social image at `assets/og-image.png`
- Real privacy policy text in `privacy.html`
- Headline, feature copy, and FAQ answers
